# Flood Control Analysis Dashboard

## Overview
This project analyzes government flood control projects in the Philippines using a dataset from Kaggle. The dashboard aims to uncover potential corruption, promote transparency, and provide actionable insights for citizens and policymakers.

## Dataset
The dataset is sourced from Kaggle:
[DPWH Flood Control Projects](https://www.kaggle.com/datasets/bwandowando/dpwh-flood-control-projects/data)

It contains information about flood control projects managed by the Department of Public Works and Highways (DPWH), including project names, locations, budgets, and completion status.

## Project Goal
- Uncover signs of corruption in government flood control projects
- Increase transparency and accountability
- Enable users to explore, filter, and visualize project data
- Provide recommendations for improvement and citizen engagement

## Project Structure
- `app.py`: Streamlit dashboard application for data analysis and visualization
- `dpwh_flood_control_projects.csv`: Kaggle dataset of flood control projects
- `requirements.txt`: List of Python dependencies
- `README.md`: Project documentation

## Usage
1. Clone the repository
2. Install dependencies:
	```
	pip install -r requirements.txt
	```
3. Run the dashboard:
	```
	python app.py
	```
